# 🧠 Curso de Programação Orientada a Objetos (POO)

Bem-vindo ao repositório dedicado ao estudo e prática dos conceitos de **Programação Orientada a Objetos (POO)**.  
Aqui você encontrará códigos, exercícios, anotações e projetos desenvolvidos ao longo do curso.

---

## 📚 Conteúdo do Repositório

- **Introdução à POO**
  - Classes e Objetos  
  - Atributos e Métodos  
  - Construtores  

- **Princípios Fundamentais**
  - Encapsulamento  
  - Herança  
  - Polimorfismo  
  - Abstração  

- **Modificadores de Acesso**
  - public, private, protected  

- **Composição e Agregação**

- **Interfaces**

- **Tratamento de Exceções**

- **Exercícios Práticos**

---

## 🛠️ Tecnologias Utilizadas

- **Java 21 LTS**  
- IDE recomendada: *IntelliJ IDEA*

---

## 🚀 Objetivo do Curso

Este repositório tem como objetivo:

- Registrar minha evolução no estudo de POO  
- Compartilhar exemplos e exercícios  
- Criar uma base para futuros projetos usando os conceitos aprendidos  
- Ajudar outros estudantes com código limpo e comentado
